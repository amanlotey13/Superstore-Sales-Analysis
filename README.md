# Superstore Sales Analysis Dashboard

## Project Overview
Interactive Power BI dashboard for analyzing sales, profit, customer segments, and regional performance.

## Tools Used
- Power BI
- Excel

## Key Objectives
- Monitor sales performance
- Analyze profit trends
- Compare regional performance
- Track KPIs

## Key Insights
- Top-performing regions identified
- Profitability trends analyzed
- Customer segment performance evaluated
